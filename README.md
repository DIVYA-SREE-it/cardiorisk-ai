# ❤️ CVD Risk Prediction using ML

## 📌 Overview
This project predicts cardiovascular disease risk using clinical + wearable data.

## 🔧 Features
- Data Cleaning Pipeline
- Feature Engineering
- Model Training (LR, RF, XGBoost)
- SHAP Explainability
- Streamlit App

## 📊 Dataset
- UCI Heart Disease
- Wearable Data (steps, sleep, HR)

## 🚀 How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run app
streamlit run app/app.py

## 📈 Results
- Best Model: Random Forest
- F1 Score: ~0.85+

## 🧠 Explainability
- SHAP feature importance
- Waterfall & dependence plots

## 📁 Output
- Processed dataset exported as CSV
